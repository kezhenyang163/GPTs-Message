### **绑定流程**
1. **登录 OpenAI**  
   访问 [OpenAI 平台](https://platform.openai.com/) 并登录你的账户。

2. **进入账户设置**  
   点击“Settings（设置）” -> “Billing（账单）”。

3. **添加付款方式**  
   选择“Add payment method（添加付款方式）”，然后输入 WildCard 虚拟信用卡的卡号、有效期、CVV 和账单地址。
